
# 💰 Django-приложение для учёта движения денежных средств (ДДС)

## 🚀 Инструкция по запуску проекта "для чайников"

### 📦 Установи нужные программы:
1. Python (рекомендуется 3.10+) — https://www.python.org/downloads/
2. Git (если планируешь использовать репозиторий) — https://git-scm.com/downloads

---

### 🛠️ Шаги запуска:

#### 1. Распакуй архив 📁
- Открой `cashflow_ready.zip` и извлеки его в любую удобную папку.

#### 2. Открой терминал в папке проекта:

**Windows:**
- Зажми `Shift + ПКМ` → "Открыть окно PowerShell здесь".

**Mac/Linux:**
```bash
cd путь_до_папки_проекта
```

#### 3. Создай виртуальное окружение:
```bash
python -m venv venv
```

#### 4. Активируй окружение:

**Windows:**
```bash
venv\Scripts\activate
```

**Mac/Linux:**
```bash
source venv/bin/activate
```

#### 5. Установи зависимости:
```bash
pip install -r requirements.txt
```

#### 6. Примени миграции базы данных:
```bash
python manage.py migrate
```

#### 7. Создай суперпользователя:
```bash
python manage.py createsuperuser
```

#### 8. Запусти сервер:
```bash
python manage.py runserver
```

#### 9. Открой в браузере:
```
http://127.0.0.1:8000/
```

---

### 🔐 Админ-панель:
Перейди по адресу: http://127.0.0.1:8000/admin  
Введи логин/пароль, который указал при создании суперпользователя.
